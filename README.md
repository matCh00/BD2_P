+ hasło "162534"

### MySQL Workbench
+ połącz się z bazą danych
    + wybierz bazę danych
    + lub: Database -> Connect to Database
+ jeżeli nie ma połączenia:
    + Server -> Startup/Shutdown -> Start Server

+ cd server  ->  npm run devStart (skrypty w package.json)
